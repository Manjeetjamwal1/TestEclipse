Yes, sanity testing can be considered a subset of regression testing
because both are performed after code changes. However, sanity testing
focuses only on the changed functionality, whereas regression testing 
verifies that the changes have not affected other existing functionalities.

Which fixture is automatically provided by Playwright Test and represents the browser tab?
Explanation

The page fixture represents a single browser tab and is automatically provided by Playwright Test.

Example:

import { test, expect } from '@playwright/test';

test('Login Test', async ({ page }) => {
  await page.goto('https://example.com');
  await expect(page).toHaveTitle(/Example/);
});

Here:

browser → Browser instance (Chromium, Firefox, WebKit)
context → Browser context (similar to an incognito session)
page → A single browser tab
locator → Used to locate elements on the page
